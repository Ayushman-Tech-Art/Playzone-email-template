# 🎮 Playzone Gaming Email Template

A responsive **gaming promotional email / landing page template** built using **HTML and CSS**.  
This project demonstrates a modern **dark gaming theme UI** with product promotions, combo offers, and trending games.

## ✨ Features

- 🎮 Dark gaming themed UI
- 📱 Responsive layout
- ⚡ Animated CTA buttons
- 🛒 Product promotion sections
- 💰 Combo deal pricing layout
- 🎯 Trending games showcase
- 📧 Email-style table layout
- 🔗 Real product links and images

1. **Header**
   - Playzone Store branding
   - Gaming tagline

   - Featured gaming keyboard

   - Price and CTA button

   - Gaming monitor showcase

4. **Combo Offer**
